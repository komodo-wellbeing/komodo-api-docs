To access the API you need to obtain an integration API key in Komodo. This can be done by
either:

-   Contacting Komodo to issue an organisation-based API key.
-   Generating an organisation-based API key in Komodo via Settings > Integrations > Create an Integration.

Any requests to the REST & Cube APIs must include an `Authorization` header, with the value set to the returned token.

## Request

<ApiRequest
    type="POST"
    url="/rest/v1/auth/login"
    request={{ headers: [{ name: '__x-api-key__', description: 'Organisation-based API key issued by Komodo' }] }}

        {
            status: 200,
            description:

        },

    "error": "Missing API key",
}`,
        },
        {
            status: 401,
            description: 'Invalid API key',
            color: 'red',
            example: `{
    "error": "Invalid API key",
}`,
        },
    ]}
/>
