Any requests to the API must include an `Authorization` header, with the value set to the returned token.

## Request

<ApiRequest
    type="POST"
    url="/rest/v1/auth/login"
    request={{ headers: [{ name: 'x-api-key', description: 'Organisation-based API key issued by Komodo' }] }}
    responses={[
        {
            status: 200,
            description:
                'A successful request will return a short-lived token that can be used to retrieve information from both our REST & Cube APIs',
            example: `{
    "token": "eyJhbGciOiJ....."
}`,
        },
        {
            status: 400,
            description: 'Missing API key',
            example: `{
    "error": "Missing API key",
}`,
        },
        {
            status: 401,
            description: 'Invalid API key',
            example: `{
    "error": "Invalid API key",
}`,
        },
    ]}
/>
